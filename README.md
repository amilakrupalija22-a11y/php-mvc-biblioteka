📚 Biblioteka knjiga

## Opis projekta
Ova web aplikacija je **MVC biblioteka knjiga** izrađena u PHP-u i MySQL-u.  
Omogućava upravljanje knjigama i kategorijama koristeći **CRUD funkcionalnosti**.  
Aplikacija koristi **Bootstrap** za responzivan dizajn i **JavaScript** za dodatne funkcionalnosti poput pretrage i potvrde brisanja.

---

## Tehnologije
- PHP (Model-View-Controller arhitektura)  
- MySQL baza podataka  
- HTML / CSS / Bootstrap 5  
- JavaScript  

---

## Funkcionalnosti

### Knjige
- Pregled svih knjiga u tabeli  
- Dodavanje novih knjiga  
- Uređivanje postojećih knjiga  
- Brisanje knjiga (sa potvrdom)  
- Pretraga knjiga po naslovu, autoru ili kategoriji

### Kategorije
- Pregled svih kategorija knjiga
